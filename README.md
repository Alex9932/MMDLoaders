## MMD Loaders

Eclipse project. Last build: Release/rgEngine_test

MMD loaders for rgEngine.

Supported file formats (READ ONLY):
- pmd (Polygon Model Data)
- pmx (Polygon Model eXtended)
- vmd (Vocaloid Motion Data)
- vpd (Vocaloid Pose Data)
